Smart Academic Management System

A full-stack Django web application for managing students, courses, and enrollments.

Live Demo
**https://vivekbastine.pythonanywhere.com**

Features
- User Registration and Login
- Complete Student CRUD Operations (Create, Read, Update, Delete)
- Dashboard with statistics
- Responsive design using template inheritance
- Django Admin Panel for backend management

Technologies Used
- Django 6.0
- SQLite Database
- HTML, CSS

How to Run Locally
1. `python manage.py runserver`
2. Register a new account
3. Login and manage students
